Start a Talk With a Trusted Adult

Use this script if starting is hard: “I need help with something and I don’t want to handle it alone.”

Step 1

Pick the Right Person

Choose someone calm, reliable, and available now.

Step 2

Keep It Specific

Share one recent example and one thing you need today.

Step 3

Set Next Action

Agree on one follow-up call, meeting, or check-in time.
